# JokeAPI - Ballerina

This is a Ballerina implementation of the JokeAPI.

- Build and run locally with Ballerina installed:

```bash
bal run
```

- Build Docker image and run:

```bash
./build_docker.sh
./run_docker.sh
```

Server listens on port 8000 and exposes `/jokes` endpoints compatible with the project's tests.
